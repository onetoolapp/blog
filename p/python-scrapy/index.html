Python Scrapy 爬虫框架入门

Python 环境搭建和 Scrapy 爬虫框架的基本使用方法。

2025-01-03
阅读时长: 1 分钟

简介

Scrapy 是一个快速的高级网页爬取和网页抓取框架，用于爬取网站并从其页面中提取结构化数据。它可以用于广泛的用途，从数据挖掘到监控和自动化测试。

本文记录 Python 环境搭建和 Scrapy 爬虫框架的基本使用方法。

环境

项目使用 Poetry 进行包管理。

Poetry 是一个现代化的 Python 包管理工具，用于管理 Python 项目的依赖关系。

参照官方文档安装即可

from pathlib import Path

import scrapy


class QuotesSpider(scrapy.Spider):
    name = "quotes"

    def start_requests(self):
        urls = [
            "https://quotes.toscrape.com/page/1/",
            "https://quotes.toscrape.com/page/2/",
        ]
        for url in urls:
            yield scrapy.Request(url=url, callback=self.parse)

    def parse(self, response):
        page = response.url.split("/")[-2]
        filename = f"quotes-{page}.html"
        Path(filename).write_bytes(response.body)
        self.log(f"Saved file {filename}")
